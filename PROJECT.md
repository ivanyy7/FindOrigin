Спецификация Telegram-бота FindOrigin:

1. Назначение программы

FindOrigin — это Telegram-бот, который получает текст или ссылку на пост и пытается найти источник этой информации.

2. Стек

- Next.JS
- Деплой: Vercel
- Внешний поиск: DuckDuckGo (парсинг HTML, без ключей)
- AI: OpenRouter (модель можно задать через переменную окружения)

Webhook
- Обработчик должен принимать POST от Telegram (update)
- Достать chat.id и text из update.message
- Отправить ответ через Telegram API методом sendMessage
- Вернуть 200 OK быстро (без долгих операций)

3. Рабочий процесс
3.1 Ввод данных

Пользователь:

- вводит текст или
- даёт ссылку на Telegram-пост.

3.2 Обработка данных

Программа:

Берёт текст или извлекает текст из поста.
Выделяет:
- ключевые утверждения
- даты
- числа
- имена
- ссылки

3.3 Ищет возможные источники:
- официальные сайты
- новостные сайты
- блоги
- исследования

Поиск реализован через внешний поисковик (DuckDuckGo): бот делает HTTP-запрос к HTML-странице поиска, парсит результаты (URL, заголовок, сниппет) и получает 5–10 кандидатов.

4. С помощью AI (через OpenRouter) сравнивает смысл, а не текст буквально:
- на вход модели подаётся текст пользователя и список найденных кандидатов;
- модель выбирает 1–3 наиболее релевантных источника и формирует краткое обоснование;
- бот возвращает пользователю ссылки и оценку уверенности (0–100%).

> Примечание: поддержка Google Custom Search JSON API (через GOOGLE_CSE_API_KEY / GOOGLE_CSE_CX) предусмотрена в коде как опциональное улучшение для окружений с настроенным биллингом в Google Cloud, но для работы проекта не требуется.
