# Промт: формат работы для всего проекта

Этот файл задаёт, как ведётся работа в **этом проекте**: формат чата и общие правила. Файл лежит в папке проекта и с ним работаем дальше.

---

## 1. Формат общения в чате

- В этом проекте используется **формат общения в чате** из файла **ПРОМТ_ФОРМАТ_ЧАТА.md** (тот же каталог или твоя копия с ПК).
- Речь идёт только об **обычном чате** с ИИ (Composer / Chat). Чаты с Агентом здесь не рассматриваются.
- Каждый **новый обычный чат** в этом проекте ведётся по этому формату: блоки КОМАНДЫ/ПОЯСНЕНИЯ, по шагам, без лишних деталей, команды — «чистые» для PowerShell.
- Чтобы не копировать промт вручную, в проекте включено правило Cursor (`.cursor/rules/`): ИИ подхватывает формат автоматически.

---

## 2. Общие правила проекта

*(Ниже можно дополнять правила, которые относятся ко всему проекту, а не только к чату.)*

- Язык общения с пользователем и в коде/комментариях: **русский**.
- Команды и примеры в терминале: **PowerShell (Windows)**; не использовать bash/Linux-синтаксис.
- Названия коммитов: **по-русски**.

- Этот файл остаётся в папке проекта.
- При добавлении новых правил проекта — дописывать их в раздел «Общие правила проекта» или в новые разделы этого файла.
